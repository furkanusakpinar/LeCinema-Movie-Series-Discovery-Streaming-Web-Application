# 🎬 LeCinema - Modern Movie & Series Discovery Platform

LeCinema is a web application that presents real-time movie and TV series data using the TMDb API with a modern, fast, and user-friendly interface.

## ✨ Key Features

**📽️ Rich Content Lists:** Instantly discover Popular, Trending, and Top Rated movies and TV shows.

**🔍 Detailed Information:** Access cast members, budgets, box office revenues, and similar content recommendations.

**🎥 Watch Trailers:** Watch movie trailers directly within the app thanks to YouTube integration.

**📱 Full Compatibility:** A responsive design optimized for mobile, tablet, and desktop devices.

**⚡ High Performance:** Fast loading times powered by Vite and React.

---

## 🛠️ Technology Stack

**Frontend:** React 19, Vite, Vanilla CSS

**API & Data:** Axios, TMDb (The Movie Database) API

**Navigation:** React Router Dom

**Icons:** Lucide & Custom SVG
